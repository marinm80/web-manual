HTML Table Reference

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables
<thead> Table Head The set of rows defining the column headers in a table
<tbody> Table Body The set of rows containing actual table data
<tr> Table Row The table row container
<td> Table Data The table data cell container
<th> Table Header The table header cell container
<tfoot> Table Foot The set of rows defining the footer in a table
<caption> Table Caption Provides a title or description for the table
<colgroup> Column Group Groups one or more columns for formatting
<col> Column Defines column properties for each column within a colgroup

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span
rowspan Row Span Defines how many rows a td element should span
scope Header Scope Specifies whether a header cell is a header for a column, row, or group
headers Header IDs Lists header cells that provide information for the current cell
abbr Abbreviation Provides an abbreviated version of the content in a header cell
align Alignment Specifies the horizontal alignment of content (deprecated in HTML5)
valign Vertical Alignment Specifies the vertical alignment of content (deprecated in HTML5)
width Width Specifies the width of a table or cell (deprecated in HTML5)

HTML Semantic Tags

Tag Name Description
<header> Header Represents introductory content or navigational aids
<nav> Navigation Defines a set of navigation links
<main> Main Content Represents the main content of the document
<article> Article Represents a standalone piece of content
<section> Section Represents a thematic grouping of content
<aside> Aside Represents content aside from the main content
<footer> Footer Represents footer information for its section
<figure> Figure Represents self-contained content like images or diagrams
<figcaption> Figure Caption Provides a caption for a figure element
<time> Time Represents a specific period in time or date
<mark> Mark Represents highlighted or marked text
<details> Details Creates a disclosure widget for additional information
<summary> Summary Provides a summary or caption for a details element

CSS Properties, Flexbox & Grid

Property Category Description
display: flex Flexbox Creates a flexible layout container
justify-content Flexbox Aligns flex items horizontally (main axis)
align-items Flexbox Aligns flex items vertically (cross axis)
flex-direction Flexbox Defines the direction of flex items (row, column)
flex-wrap Flexbox Controls whether flex items wrap to new lines
display: grid Grid Creates a two-dimensional grid layout
grid-template-columns Grid Defines the columns of the grid
grid-template-rows Grid Defines the rows of the grid
grid-gap Grid Sets the gap between grid items
grid-area Grid Specifies a grid item's size and location
position CSS Layout Specifies the positioning method (static, relative, absolute, fixed)
box-sizing CSS Layout Defines how the total width and height is calculated
z-index CSS Layout Controls the stacking order of positioned elements
transform CSS Effects Applies 2D or 3D transformations to elements
transition CSS Effects Creates smooth transitions between property changes
